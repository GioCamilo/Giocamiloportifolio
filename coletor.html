Project PDF to Excel data converter

About the project

Application developed entirely in Python as an automation solution to assist a department in extracting and filtering data from documents received on a monthly basis. The process is simple and efficient: the user selects the PDF documents, waits for the automatic data extraction, reviews the extracted information, and exports the result to a pre-formatted Excel spreadsheet ready for use. The tool drastically reduces the time spent on manual tasks, ensuring greater speed, accuracy, and consistency in reports.

Libraries and tools:
Pandas Openpyxl Tkinter Excel PyPDF2
